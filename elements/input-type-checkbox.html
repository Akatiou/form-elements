Input Type checkbox

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="checkbox"> est un élément de saisie qui permet de sélectionner une valeur dans un formulaire grâce à une case à cocher.

Exemple

glace vanille chantilly chocolat chaud biscuit

Code derrière tout ça :

<input type="checkbox" name="nom" value="valeur au  bouton" >

Quelques attributs :

  • checked : Cet attribut booléen indique si le contrôle est sélectionné ou non. On peut également utiliser checked="checked"pour indiquer l'état coché.
  • value : Cet attribut indique la valeur initiale du contrôle. Si cet attribut est absent, la valeur par défaut sera la chaîne de caractères "on".
  • indeterminate : Cet attribut interne (IDL) est utilisé pour indiquer que la case à cocher est dans un état indéterminé (cela se traduit généralement par un trait horizontal sur la case). Il n'est pas directement manipulable en HTML.
