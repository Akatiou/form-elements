Input Type email

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="email"> est un élément de saisie qui permet à l'utilisateur d'entrer et de modifier une ou plusieurs adresses e-mails, si l'attribut multiple est spécifié. La valeur d'entrée est automatiquement validée pour s'assurer qu'elle ne soit pas vide ou que l'adresse courriel saisie soit correcte, avant que le formulaire ne puisse être envoyé. Les pseudo-classes de CSS :valide et :invalide sont automatiquement appliquées selon le cas.

Exemple

Code derrière tout ça :

<input type="email" id="emailAdress">
<input type="email" id="courriel">
<input id="emailAddress" type="email" multiple >

Quelques attributs :

  • Type : l'attribut le plus important est l'attribut type. Ce dernier est extrêmement important puisqu'il définit le comportement de l'élément < input >. Il peut radicalement changer le sens de l'élément, faites-y attention.
  • Placeholders : Parfois, il est utile d'offrir un indice sur le contexte de la forme que les données d'entrée devraient prendre. Ceci peut être particulièrement important si la conception de la page n'offre pas d'explications descriptives pour chaque input>. C'est là que les "placeholders" sont utiles. Un "placeholders" est une valeur qui démontre la forme que la valeur devrait prendre en présentant un exemple valide, qui s'affiche dans la zone d'édition. Une fois que les données sont entrées dans la boîte, l'espace réservé disparaît; Si la case est vidée, l'espace réservé réapparaît
  • Multiple : En ajoutant l'attribut multiple, l'entrée peut être configurée pour accepter plusieurs adresses de messagerie. Maintenant, la saisie est considérée comme valide lorsqu'une seule adresse e-mail est entrée, ou lorsqu'un nombre d'adresses email séparées par des virgules et, éventuellement, un certain nombre de caractères blancs sont présents.
